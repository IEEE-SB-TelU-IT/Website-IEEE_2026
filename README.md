# IEEE SB Telkom University Website 🚀

Website resmi **IEEE Student Branch Telkom University** yang dirancang dengan desain modern, responsif, dan interaktif menggunakan React.js dan Tailwind CSS.

## 🛠️ Tech Stack
- **Frontend**: [React.js](https://reactjs.org/)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **Icons**: [Lucide React](https://lucide.dev/)
- **Build Tool**: [Vite](https://vitejs.dev/)
